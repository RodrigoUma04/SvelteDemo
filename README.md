# Svelte & SvelteKit Demo Presentation

Een interactieve demo website gebouwd met SvelteKit om Svelte en SvelteKit te presenteren.

## Waarom een demo website?

In plaats van een traditionele PowerPoint presentatie heb ik ervoor gekozen om een **interactieve demo website** te maken met SvelteKit zelf. Dit heeft verschillende voordelen:

- **Leren door te doen**: Door de website in SvelteKit te bouwen, heb ik het framework daadwerkelijk gebruikt
- **Interactieve voorbeelden**: Bezoekers kunnen live de features uitproberen (counter, text input, etc.)
- **Toont de kracht van Svelte**: De website zelf is een bewijs van hoe snel en eenvoudig je iets kunt bouwen met Svelte
- **Beter dan statische slides**: Voor een web framework is een werkende website relevanter dan PowerPoint slides

## Wat zit erin?

De website bevat 4 hoofdsecties:

1. **Quick Comparison** - Verschil tussen Svelte en SvelteKit
2. **Svelte Reactivity** - Interactieve demo's van Svelte's reactivity
3. **SvelteKit Features** - Overzicht van alle SvelteKit functionaliteit
4. **Code Examples** - Real-world code voorbeelden

## Installatie & Gebruik
```bash
# Clone de repository
git clone git@github.com:RodrigoUma04/SvelteDemo.git
cd SvelteDemo

# Installeer dependencies
npm install

# Start development server
npm run dev
```

Open `http://localhost:5173` in je browser.
